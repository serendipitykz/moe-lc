位置编码与长度外推

预训练长度之外的 token 位置在标准 PE 下"越界"——位置编码外推是长上下文的关键技术。

RoPE

旋转位置编码

将 token 位置编码为向量旋转角度：

\[\theta_i = 10000^{-2i/d}\]

点积天然体现相对位置：\(\langle R_m q,\, R_n k \rangle\) 仅依赖 \(m - n\)

默认外推性差；Position Interpolation (PI) 通过缩放 \(\theta\) 扩展范围。

Used in: LLaMA, Mixtral, DeepSeek

ALiBi

Attention with Linear Biases

对 attention logit 直接加线性惩罚（无位置向量）：

\[\text{score}_{ij} = q_i k_j^{\top} - m \cdot |i - j|\]

\(m\) 为 head 专属斜率（等比数列）；距离越远惩罚越大，自然外推。

Used in: BLOOM, MPT

YaRN

Yet another RoPE extensioN

NTK-aware 缩放：高频分量插值，低频分量外推。

Scale factor ≈ 8，将 context 从 4K 扩展至 128K。

微调成本低——仅需 ~400 步长文档训练。

Used in: Mistral 7B, extended LLaMA

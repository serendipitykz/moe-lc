Mixture of Depths 是最有潜力的方向之一，简短介绍即可。重点是让听众感受到这个领域还在快速发展。

未来方向展望

新范式

Mixture of Depths (MoD)

概念：不仅对"宽度"（哪些专家）稀疏化， 也对"深度"（哪些层）稀疏化

动态跳过 token 不需要的层 → 每个 token 自适应选择计算深度

[Raposo et al., 2024] 在固定 FLOPs 预算下超越 isoFLOP 基线

可微路由

软路由（Soft MoE）

问题：TopK 路由不可微 → 梯度估计有噪声，训练不稳定

Soft MoE：每个"slot"由 所有 token 的加权混合形成， 完全可微，梯度流动顺畅

[Puigcerver et al., 2023] 在 ViT 上验证，训练稳定性显著提升

Trade-off

无法 exactly 将单个 token 路由到单个专家， token 边界模糊 → 不适合需要 token-level 追踪的任务

多模态

多模态 MoE

自然延伸： 视觉、语言、代码各有"领域"→ 模态感知路由成为可能

图像 patch 和文本 token 可路由到不同专家， 实现模态间的自然隔离与协作

前景：MoE 可能是 统一多模态模型 的自然架构选择

系统

硬件协同设计

MoE 的 all-to-all 通信和专家 offloading 是当前核心瓶颈

下一代 NVLink / CXL 高带宽互联将使更大 \(N_{\text{exp}}\) 成为可能

专为 MoE 设计的 HBM 分层管理 （专家 cache 层次结构）

芯片内集成路由逻辑， 减少 dispatch 延迟，使稀疏激活真正"零开销"

Top-K 路由机制

Gating Function

\[ \mathbf{g}(x) = \text{TopK}\!\left(\text{Softmax}(W_g \cdot x),\; k\right) \]

Layer Output

\[ y = \sum_{i \in \mathcal{T}_k} g_i(x) \cdot \text{FFN}_i(x) \]

其中 \(\mathcal{T}_k\) 为 top-k 选中的专家下标集合

每个 token 独立计算门控分数，仅激活得分最高的 k 个专家，其余专家参数不参与当前 token 的前向计算。

⚠ 非激活专家的梯度：在反向传播中也不更新，可能导致专家分化不均
